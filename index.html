DeleteTweets

Bulk-delete your old X.com posts, replies & retweets — right from your browser. No signup, no API keys, no data shared.

What it does

📝 Deletes tweets
💬 Deletes replies
🔁 Removes retweets
📅 Filters by age
📊 Live progress UI
🛑 Stop anytime

Scans your timeline page-by-page using X.com's own internal API, filters posts older than 30 days (configurable), and deletes them one by one with rate-limit-friendly delays. A dark overlay shows live progress.

⚙️ Configuration

      
To change how far back to delete, edit the top of delete-tweets.js:

      

        
const DAYS_OLD = 30;                // posts older than 30 days
const DELAY_BETWEEN_DELETES = 1200; // ms between API calls
const MAX_ERRORS = 15;              // stop after N consecutive errors
